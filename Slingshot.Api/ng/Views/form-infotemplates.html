ARM Templates

When a user clicks on the "Deploy to Azure" button, we load a default Azure Resource Manager (ARM) template which can be used to create a site with code deployed from your GitHub repository. If you would like to customize the template used, you can add a file called "azuredeploy.json" next to your README.md file. Properties that are exposed through your template will show up as input parameters in the site.

Click here for an example MVC GitHub project which uses a custom template. Or for more detailed information on using ARM templates with the Deploy to Azure button, check out this blog post.
